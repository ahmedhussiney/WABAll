Info Summary widget

The Info Summary widget allows you to provide a count of features contained in the current map extent for each layer specified. Each layer on the widget panel can be expanded to show a list of features in the current extent, optionally grouped by a specified field. Point layers in the widget can be configured to display as clusters.

Configuring the Info Summary widget

The Info Summary widget can be set to open automatically when apps start. To do so, click the dot on the widget to turn it dark green.

Steps:
  1. Hover over the widget and click the small edit icon . The configuration window for this widget opens.
  2. Optionally click the change widget icon button and replace it with your own image.

    A file explorer window opens, allowing you to select a local image file to use as the widget icon.

  3. Click Add Layer. Each added layer specifies a feature layer in the map to be counted and displayed by the widget. The following layer types are supported with the Info Summary widget:
    • Feature services
    • Feature collections
    • Map services
    • Comma-separated values (CSV) files (.csv)
    • Keyhole markup language (KML) file (.kml, .kmz)
    • GeoRSS web feed (URL)
  4. Optionally add a label for each layer.
  5. By clicking the small edit icon under the Display Options header, you can update the symbology options and panel display options.
    1. Use the Symbol tab to change the map symbology (symbol drawn on the map), enable clustering for point layers, or update the panel symbology (symbol displayed in the widget panel).
    2. Use the Panel tab to manage how the list of features displays on the widget panel. Provide a field and label under Layer Options to display features by a specified field. Check on the Group Features checkbox under Group Options to group features in the list by a specified field.
  6. Optionally select Refresh for Feature Collection layer types to continually update the data at a specified interval. If the Refresh option is enabled, a refresh interval is required.
  7. Under Actions you can change the order of layers or delete a layer.
  8. Optionally select Display Feature Counts to display a count of features for each layer on the widget panel. The count will display the number of features in the current map extent for the selected layer.
  9. Optionally select Display Logo Image to add text or an image to the widget header. Click the Upload Image button to add your own image.

    A file explorer window opens, allowing you to select a local image file.

  10. Specify text in the Main Panel Text box to display below the logo image. An example would be “Last Updated:”.

Using the Info Summary widget

Steps:
  1. Click the Info Summary widget icon to open it. The widget displays on the right side of the map.
  2. All configured layers are turned on by default and those with clustering enabled will display as clusters.
  3. Click layer icons to toggle the layers off and on.
  4. Click a layer on the widget panel to expand the list of features within the current map extent. If the layer is grouped by a specified field, use the arrow to expand it, showing a list of features within the selected group.
  5. Pan and zoom around the map to update the feature counts on the widget panel.
  6. Click the Info Summary widget or the X in the upper right corner of the widget to close it. When the widget closes, the cluster layers will remain visible. To turn the clusters off, uncheck them in the Layer List widget.
